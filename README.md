# DS-Benchmark

Бенчмарк-пайплайн для сравнения реализаций теории Демпстера-Шейфера через единый интерфейс адаптеров.

## Доступные адаптеры

Адаптеры подключаются напрямую в `scripts/profile_benchmark.py` (без отдельного глобального реестра),
а внешние реализации разнесены по отдельным модулям в `src/adapters/external/`.

- `our` — внутренняя реализация из `src/core/dempster_core.py`.
- `pyds` — адаптер для сторонней библиотеки `pyds`.
- `dempster_shafer` — адаптер для сторонней библиотеки `dempster_shafer`.

Доступность конкретного внешнего адаптера определяется автоматически по установленным пакетам.

## Запуск

```bash
python scripts/profile_benchmark.py --library our --tests last
```
