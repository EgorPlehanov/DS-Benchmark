# Data / Данные

## RU
Каталог входных данных для тестов и валидации.

### Содержимое
- `reference_template.json` — шаблон структуры тестового кейса.
- `book_examples/` — примеры из книги для функциональной проверки.
- (опционально) `generated/` — автоматически сгенерированные тесты.

### Формат (DASS)
Ожидаются поля:
- `frame_of_discernment`
- `bba_sources`
- `metadata` (рекомендуется)

---

## EN
Input data catalog for tests and validation.

### Contents
- `reference_template.json` — test-case structure template.
- `book_examples/` — book-based examples for functional validation.
- (optional) `generated/` — auto-generated test sets.

### Format (DASS)
Expected fields:
- `frame_of_discernment`
- `bba_sources`
- `metadata` (recommended)
